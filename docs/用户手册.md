# 用户使用手册

## 欢迎使用视频剪辑助手！

本软件是一款专业的视频拆分与合成工具，支持硬件加速，操作简单，性能强大。

---

## 快速开始

### 第一步: 打开视频文件

1. 点击顶部工具栏的 **"打开文件"** 按钮
2. 或使用菜单: `文件` → `打开视频`
3. 或使用快捷键: `Ctrl+O`
4. 选择您要处理的视频文件 (支持 MP4, AVI, MKV, MOV, FLV)

### 第二步: 查看视频信息

打开视频后,左侧信息面板会显示:
- **分辨率**: 视频的宽度和高度 (如 1920x1080)
- **帧率**: 每秒帧数 (如 25 fps)
- **码率**: 视频比特率 (如 2000 kbps)
- **总帧数**: 视频包含的总帧数
- **时长**: 视频总时长
- **编码格式**: 视频编码方式 (如 h264)

### 第三步: 预览视频

- 点击 **"播放"** 按钮开始播放
- 拖动进度条快速跳转到任意位置
- 再次点击变为 **"暂停"**

## 主要功能

### 🎬 功能一: 拆分视频

将视频拆分为图片序列和音频文件。

**操作步骤**:

1. **打开视频**: 使用"打开文件"按钮加载视频

2. **开始拆分**: 点击工具栏的 **"拆分视频"** 按钮

3. **选择输出目录**: 选择一个文件夹保存拆分结果

4. **等待完成**: 
   - 底部进度条会显示拆分进度
   - 状态栏显示当前操作

5. **查看结果**: 拆分完成后,输出目录包含:
   ```
   输出目录/
   ├── frames/           # 图片序列文件夹
   │   ├── frame_000001.jpg
   │   ├── frame_000002.jpg
   │   └── ...
   └── audio.mp3         # 提取的音频文件
   ```

**应用场景**:
- 提取视频中的所有画面进行编辑
- 单独使用视频中的音频
- 制作视频缩略图

---

### 🎞️ 功能二: 合成视频

将图片序列和音频合成为完整视频。

**操作步骤**:

1. **准备素材**:
   - 一个文件夹,包含按顺序命名的图片 (如 frame_000001.jpg, frame_000002.jpg...)
   - 一个音频文件 (MP3, WAV, AAC等)

2. **开始合成**: 点击工具栏的 **"合成视频"** 按钮

3. **选择图片文件夹**: 选择包含图片序列的文件夹

4. **选择音频文件**: 选择要合并的音频文件

5. **保存位置**: 选择输出视频的保存路径和文件名

6. **等待完成**: 
   - 进度条显示合成进度
   - 合成完成后弹出提示

**注意事项**:
- 图片应具有相同的分辨率
- 图片命名应有序 (软件会按文件名排序)
- 音频长度应与图片数量匹配,否则会被截断或静音填充

**应用场景**:
- 将编辑后的图片序列合成视频
- 为图片幻灯片添加背景音乐
- 制作定格动画

---

### 📸 功能三: 设置封面

从视频中提取任意一帧作为封面图片。

**操作步骤**:

1. **播放或跳转**: 将视频播放到想要的位置
   - 可以使用播放按钮
   - 也可以拖动进度条

2. **设置封面**: 点击 **"设为封面"** 按钮

3. **保存图片**: 选择保存位置和文件名
   - 支持 JPEG (.jpg) 和 PNG (.png) 格式

**应用场景**:
- 制作视频缩略图
- 提取精彩瞬间
- 创建视频海报

## 技术规格

### 支持的视频格式

**输入格式**:
- MP4 (.mp4)
- AVI (.avi)
- MKV (.mkv)
- MOV (.mov)
- FLV (.flv)
- 其他FFmpeg支持的格式

**输出格式**:
- MP4 (H.264编码)

### 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)

### 支持的音频格式

- MP3 (.mp3)
- WAV (.wav)
- AAC (.aac)
- M4A (.m4a)

### 硬件加速

本软件支持以下硬件加速:
- **NVIDIA GPU**: 使用 NVENC/NVDEC
- **Intel GPU**: 使用 Quick Sync Video
- **AMD GPU**: 使用 AMF

如果没有可用的硬件加速,将自动切换到高效的软件编解码。

## 常见问题 (FAQ)

### Q1: 为什么视频加载失败?

**可能原因**:
1. 视频文件损坏
2. 不支持的编码格式
3. 文件路径包含特殊字符

**解决方法**:
- 尝试使用其他视频文件
- 将视频转换为MP4格式
- 确保文件路径只包含英文和数字

---

### Q2: 拆分视频速度慢怎么办?

**优化建议**:
1. 关闭其他占用GPU的程序
2. 选择较低的输出质量
3. 确保硬盘有足够空间

---

### Q3: 合成视频没有声音?

**检查项**:
1. 音频文件是否有效
2. 音频格式是否支持
3. 查看错误提示信息

---

### Q4: 图片序列命名规则?

**推荐格式**:
- `frame_000001.jpg`
- `image_0001.png`
- `pic001.jpg`

**关键点**:
- 使用数字编号
- 保持位数一致 (用0填充)
- 按字母顺序排列

---

### Q5: 能处理多大的视频?

**理论上无限制**,但受限于:
- 可用内存
- 硬盘空间
- 处理时间

**建议**:
- 视频时长: < 1小时
- 分辨率: ≤ 4K (3840x2160)
- 文件大小: < 10GB

## 性能提示

### 💡 提升处理速度

1. **使用SSD**: 将输入/输出文件放在固态硬盘
2. **升级硬件**: 更好的CPU/GPU可显著提升速度
3. **降低质量**: 选择较低的输出分辨率或帧率
4. **关闭预览**: 处理时关闭预览窗口

### 💡 节省磁盘空间

1. **JPEG质量**: 拆分时选择合适的JPEG质量 (70-85)
2. **视频码率**: 合成时使用适当的码率 (不必过高)
3. **及时清理**: 处理完成后删除临时文件

## 系统要求

### 最低配置

- **操作系统**: Windows 10/11 (64位)
- **处理器**: Intel Core i3 或同等级别
- **内存**: 4GB RAM
- **显卡**: 支持OpenGL 2.0
- **硬盘**: 100MB 安装空间 + 足够的工作空间

### 推荐配置

- **操作系统**: Windows 11 (64位)
- **处理器**: Intel Core i5/i7 或 AMD Ryzen 5/7
- **内存**: 8GB+ RAM
- **显卡**: NVIDIA GTX 1050 / Intel UHD 630 / AMD RX 560 或更高
- **硬盘**: SSD 固态硬盘
